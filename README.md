## Installation
Make a QR Code from the mentioned website (https://au.gmented.com/app/marker/marker.php) to generate a 2D Barcode
Take number to be:
99  - Square
100 - Cone
101 - Cylinder
102 - Sphere
103 - Tetrahedron

### Prerequisites
Install Mozilla Firefox on your Mobile
